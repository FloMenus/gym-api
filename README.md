# Gym API

Projet d'API pour salles de sport avec Node.js + TypeScript + Express + Prisma + PostgreSQL.

## Comment lancer

**Copier les variables d'environnement**
```bash
cp .env.example .env
```

**Lancer le conteneur Docker**
```bash
docker compose up --build
```

## Config BDD

- Base: `gym_db`
- User: `gym_user` 
- Password: `gym_password`
- Port: `5432`
